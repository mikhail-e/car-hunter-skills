## Анализ документов (OCR)

Для работы со сканированными PDF (фото без текстового слоя) используй Tesseract OCR.

### Зависимости

```
brew install poppler tesseract tesseract-lang
```

### Пайплайн

1. **PDF → PNG**: `pdftoppm -png -r 300 <input>.pdf <output_dir>/page` (300 DPI для качественного OCR)
2. **PNG → текст**: `tesseract <page>.png <output> -l <lang>` для каждой страницы
   - `-l deu` для немецких документов
   - `-l eng` для английских
3. **Собрать текст**: объединить постранично в один файл с разделителями `========== SEITE/PAGE N ==========`
4. **Сохранить оба формата в проекте**:
   - `*_text.txt` — сырой OCR-текст (для поиска и верификации)
   - `*_text.md` — структурированный markdown (для чтения и анализа)

### Важно

- Tesseract не видит файлы из `/tmp` в sandbox — копируй изображения в директорию проекта перед OCR
- После OCR удаляй временные PNG из проекта
- В markdown-версии структурируй по страницам/секциям, используй таблицы для табличных данных
